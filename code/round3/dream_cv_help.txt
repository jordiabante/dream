Description:
    Wrapper to run cross-validation in parallel.

Usage:
	dream_cv.sh  [OPTIONS]  -- FILE_SUFFIXES

Options:
    -h--help                 	Help.
    -t--threads              	# threads. Default: 2.
    -o--outfile              	Output file name. Default: cross_validation.txt.

Examples:
    dream_cv.sh -t 5 -- file.txt

Dependancies:
    cross_validation_one_time.R

    2015/12/16
