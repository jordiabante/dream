Description:
    Wrapper to build the predictor matrix in parallel.

Usage:
	dream_precitor_matrix.sh  [OPTIONS]  -- FILE_SUFFIXES

Options:
    -h--help                 	Help.
    -t--threads              	# threads. Default: 2.
    -o--outfile              	Output file name. Default: cross_validation.txt.

Examples:
    dream_predictor_matrix.sh -t 5 -- file.txt

Dependancies:
    predictor_matrix.R

    2015/12/16
